1. 検索条件

待機中

通常はブラウザの「ダウンロード」フォルダへ保存されます。複数ファイルをまとめて落とすので、ブラウザ側で複数ダウンロード許可が必要な場合があります。

`PDF` は `law_data` / `attachment` API を使って、e-Gov の添付ファイルとして提供される公式 PDF をそのまま保存します。PDF がない法令では取得できません。

もし検索に失敗する場合は、e-Gov API の CORS 制限でブラウザから直接アクセスできない可能性があります。その場合は補助サーバー方式に切り替える必要があります。

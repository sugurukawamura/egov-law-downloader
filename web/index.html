e-Gov 法令ダウンローダー

このファイルは Mac 専用ではありません。ブラウザで開けば、Windows / Mac / Linux で使えます。検索結果から複数の法令を選び、複数形式をまとめてダウンロードできます。

1. 検索条件

待機中

通常はブラウザの「ダウンロード」フォルダへ保存されます。複数ファイルをまとめて落とすので、ブラウザ側で複数ダウンロード許可が必要な場合があります。

`PDF(XML変換)` は e-Gov 公式 PDF ではありません。XML を取得して印刷画面を開き、そこで PDF 保存する流れです。ブラウザのポップアップ許可が必要な場合があります。

もし検索に失敗する場合は、e-Gov API の CORS 制限でブラウザから直接アクセスできない可能性があります。その場合は補助サーバー方式に切り替える必要があります。
